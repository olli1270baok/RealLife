# 🎓 AdultGuide – Was dir die Schule nie beigebracht hat

Eine professionelle Web-App für junge Erwachsene. Alle wichtigen Lebensthemen kompakt und verständlich erklärt.

## Was die App enthält

| # | Thema | Inhalte |
|---|-------|---------|
| 1 | 🏠 **Eigene Wohnung** | Wohnungssuche, Mietvertrag, Kaution, Nebenkosten, Mieterrechte |
| 2 | 🛡️ **Versicherungen** | Haftpflicht, Krankenversicherung, Hausrat, Berufsunfähigkeit |
| 3 | 📊 **Steuern** | Steuererklärung, Lohnsteuer, Werbungskosten, Selbstständigkeit |
| 4 | 🏛️ **Behörden & Ämter** | Anmeldung, Arbeitsagentur, Finanzamt, Sozialamt, BAföG |
| 5 | 💰 **Finanzen & Budget** | 50/30/20-Regel, Notfallpolster, ETF-Sparpläne, Schulden |
| 6 | ⚖️ **Rechtliches** | Verbraucherrechte, Abo-Fallen, Arbeitsvertrag, Anwalt |
| 7 | 🏥 **Gesundheit** | Arztbesuche, Krankmeldung, Apotheke, Psychische Gesundheit |
| 8 | 📱 **Digital & Datenschutz** | Handyvertrag, DSL, Passwörter, DSGVO |
| + | 🌱 **Altersvorsorge** | 3-Säulen-System, ETF, Rente, FIRE-Bewegung |

## App-Features

- **Interaktive Module** – Jedes Thema mit mehreren Unterkapiteln, Navigation und Fortschrittsbalken
- **Suchfunktion** – Alle Inhalte durchsuchbar
- **Persönliche Checkliste** – 35+ Aufgaben, Fortschritt wird im Browser gespeichert
- **Schnellreferenz** – 12 Anlaufstellen mit Adressen und Hotlines
- **Glossar** – 25 Begriffe aus Behördendeutsch einfach erklärt
- **Praxis-Beispiele** – Konkrete Szenarien in jedem Themenbereich

## Dateien

- `index.html` – Hauptdatei
- `styles.css` – Dark-Mode Glassmorphism Design
- `app.js` – Alle Inhalte und App-Logik

## Öffnen

Einfach `index.html` im Browser öffnen – keine Installation, kein Server nötig!
